# Projeto de Logística de Apanha - Avicultura

Este repositório contém os scripts e dados para o projeto de otimização da logística de apanha na avicultura.

## Estrutura do Repositório

- `data/`: Contém os dados utilizados no projeto.
  - `raw/`: Dados originais e brutos. Os arquivos CSV nesta pasta devem conter as seguintes colunas: `aviario`, `nome produtor`, `latitude`, `longitude`.
  - `processed/`: Dados após o pré-processamento.
- `src/`: Contém o código fonte da aplicação.
- `notebooks/`: Contém notebooks para exploração e análise de dados.
- `docs/`: Documentação adicional do projeto.
